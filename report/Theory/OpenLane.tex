\section{OpenLane}

OpenLane is an automated RTL to GDSII flow based. The components include open-source tools, which include followings:-
\begin{enumerate}
    \item Yosys
    \item OpenROAD
    \item OpenSTA
    \item CU-GR
    \item TritonRoute
    \item DEF2SPEF
    \item Magic
    \item Netgen
    \item CVC
    \item Klayout
\end{enumerate}
